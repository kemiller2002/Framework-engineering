# Evidence Decision Records

## Purpose

Evidence Decision Records convert completed evidence into the next research action.

An EDR sits between evidence collection and any hypothesis review or next-experiment selection.

## Core Principle

Evidence Collection Runs do not directly update confidence.

Evidence must pass through an Evidence Decision Record before hypothesis review or next-experiment selection.
