# EDR Decision Rules

## Purpose

Define how evidence should be converted into next research action.

## Rules

1. Do not update confidence directly from an ECR or experiment.
2. Use observations first, interpretations second, decisions third.
3. Prefer the next experiment that most reduces the largest remaining uncertainty.
4. Prefer narrowing competing explanations over expanding theory.
5. If instrumentation ambiguity dominates, fix the instrument before scaling the claim.
6. If replication uncertainty dominates, queue replication before new abstraction.
7. If a result is surprising but fragile, queue confirmation before promotion.
8. If a result is unsurprising and low-information, deprioritize similar runs.
9. Route all confidence changes through accepted EDR output and subsequent hypothesis review.
