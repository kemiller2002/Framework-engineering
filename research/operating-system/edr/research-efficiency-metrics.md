# Research Efficiency Metrics

## Purpose

Track whether research infrastructure is increasing evidence quality or only adding overhead.

## Suggested Metrics

- time from run completion to EDR draft
- time from EDR draft to accepted decision
- number of uncertainties reduced per run
- number of redundant observations collected
- number of queue items created per EDR
- fraction of queued experiments later judged low-information

## Rule

If operating overhead rises without corresponding evidence gain, simplify the process.
