# Evidence Run Checklist

## Before Run

- hypotheses selected
- predictions written
- kill conditions identified
- packet version frozen
- prompt version frozen
- response schema frozen
- model participants listed
- data storage path created
- analysis plan written

## After Run

- raw outputs locked
- observations recorded
- interpretation recorded separately
- confidence updates proposed
- kill conditions checked
- dependency graph reviewed
- product relevance noted
- next run proposed
