# Research Operating System

## Purpose

This directory defines the operating structure for Framework Engineering research.

The goal is to organize the program around hypotheses and evidence rather than isolated one-off experiments.

## Contents

- `research-operating-manual.md`
- `hypothesis-registry.md`
- `evidence-run-template.md`
- `measurement-schema.md`
- `confidence-update-guide.md`
- `kill-condition-guide.md`
- `competing-hypotheses.md`
- `research-dashboard-template.md`
- `decision-log.md`

## Operating Principle

Every evidence run should test multiple hypotheses when practical, collect common measurements, define predictions in advance, update confidence after the run, and respect kill conditions.
