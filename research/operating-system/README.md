# Research Operating System

## Purpose

This directory defines the operating structure for Framework Engineering research.

The goal is to organize the program around hypotheses and evidence rather than isolated one-off experiments.

## Operating Principle

Every evidence run should test multiple hypotheses when practical, collect common measurements, define predictions in advance, route evidence through an Evidence Decision Record, and respect kill conditions.
