# Evidence Dashboard

This directory contains the Framework Engineering evidence dashboard for the current reviewed state of the research program.

Purpose:

- summarize completed ECRs
- summarize active hypotheses and their current reviewed status
- summarize defensible and unsupported claims
- summarize instrument state and material threats
- identify the next highest-information research action

The dashboard is generated by:

`build-dashboard.js`

It does not assign numerical confidence.
