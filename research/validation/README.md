# Validation

Status: Working draft

Validation exists to weaken, falsify, simplify, or substantially revise Framework Engineering claims before those claims are treated as durable.

Discovery is now complete enough to justify a validation phase.

The goal is not theory expansion.

The goal is to test whether the current theory survives adversarial review.

Files in this directory define the validation program, adversarial review process, competing-theory challenge, replication plan, claim review template, and validation roadmap.
