# SWOT Version B

Use the structure below to analyze the scenario.

## Scope

State the decision context and boundary of the analysis.

## Observations

List the scenario observations before classifying them.

## SWOT Classification

Classify observations as strengths, weaknesses, opportunities, or threats.

## Evidence / Confidence

Use categorical confidence only.

## Priorities

Identify which factors most need attention first.

## Recommended Actions

Recommend near-term actions that follow from the analysis.
