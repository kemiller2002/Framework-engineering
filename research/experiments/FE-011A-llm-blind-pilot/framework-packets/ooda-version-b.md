# OODA Version B

Use the structure below to support a calibrated OODA pass.

## Observe

Record the immediately relevant signals and facts.

## Evidence Status

State what is known, uncertain, or missing.

## Orientation Assumptions

Describe the assumptions shaping interpretation.

## Decision Options

List plausible response options.

## Confidence / Uncertainty

Use categorical confidence only.

## Action

State the recommended action.

## Reassessment Triggers

State what new evidence or conditions should trigger reassessment.
