# Results

Store runs in framework-specific subfolders:

results/
  swot/
  five-whys/
  ooda/

For each run, record:

- model name
- date
- framework packet
- scenario
- participant output
- evaluator output
- notes

This directory is for actual pilot outputs only.

Do not create fabricated runs or placeholder results that could be mistaken for study evidence.
