# FE-012A Reconstruction Protocol

## Purpose

Test whether primitive sequences preserve meaningful reasoning structure.

## Protocol

- Select extracted primitive sequence from an artifact.
- Remove artifact name and domain.
- Give sequence to reconstruction analyst.
- Ask analyst to reconstruct the reasoning flow in plain language.
- Compare reconstructed flow to original artifact structure.
- Record whether the reconstruction is recognizable.

## Reconstruction Ratings

- strong
- moderate
- weak
- failed
- unclear

No numeric scores.

## Interpretation Note

Reconstruction failure is useful evidence.

It may indicate missing primitives, over-normalization, weak blinding design, or that the artifact does not reduce cleanly to the current primitive set.
