# FE-013 Artifact Set

## Selected Artifacts

1. Scientific Method
2. OODA Loop
3. Scrum
4. Differential Diagnosis
5. SWOT
6. Bloom's Taxonomy

## Rationale

- Scientific Method: cyclic inquiry
- OODA: adaptive decision loop
- Scrum: iterative execution workflow
- Differential Diagnosis: branching diagnostic reasoning
- SWOT: static classification / assessment
- Bloom: hierarchical classification artifact

## Design Intent

The set deliberately mixes procedural and more static artifacts.

If the instrument is useful, richer control-flow structure should emerge more clearly for the procedural artifacts than for the static or classificatory artifacts.
