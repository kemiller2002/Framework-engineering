# FE-013 Interpretation Rules

- Do not claim frameworks are programming languages.
- Do not claim all frameworks have procedural ASTs.
- If static artifacts resist AST extraction, treat that as useful boundary evidence.
- If models force ASTs onto static artifacts, treat that as instrument bias.
- The hypothesis strengthens only if procedural and static artifacts produce meaningfully different structures.
- The hypothesis weakens if the schema forces every artifact into the same shape.
