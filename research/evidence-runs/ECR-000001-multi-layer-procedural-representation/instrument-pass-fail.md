# ECR-000001 Instrument Pass / Fail

## Success Criteria

- Valid JSON from all models.
- Schema fields are filled consistently enough for comparison.
- Static and procedural packets produce distinguishable structures.
- Product relevance stays observational rather than promotional.
- Ambiguity is recorded instead of resolved by hallucination.

## Failure Criteria

- Models update hypotheses directly.
- Models produce theory claims.
- All artifacts look structurally identical.
- Models invent unsupported structure.
- Schema complexity causes malformed outputs or empty fields.
