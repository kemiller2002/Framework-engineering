# Semantic Normalization Rules

- Normalization is deterministic.
- Raw text remains preserved.
- Ontology patterns are provisional.
- Matching is case-insensitive.
- Prefer exact substring matching first.
- If multiple concepts match, record all matches.
- If no concept matches, record the item in `unresolved-semantic-matches.md`.
- Semantic agreement is not proof of actual equivalence.
- Semantic normalization is a measurement aid, not theory validation.
