# False Positive Risk

False positives occur when different procedural concepts normalize to the same concept.

High-risk pairs:
- Termination vs Decision
- Evaluation vs Verification
- Comparison vs Evaluation
- Classification vs Sequencing
- Communication vs Coordination
- Reflection vs Reassessment
- Execution vs Validation

Why this matters:
- false positives inflate semantic agreement
- false positives are harder to notice than unresolved text
- false positives can make hypothesis review look stronger than it is
