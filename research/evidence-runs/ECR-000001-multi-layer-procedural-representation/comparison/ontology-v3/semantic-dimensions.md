# Semantic Dimensions

## Dimensions

1. Role
The procedural operation being performed.

2. Object
The thing the operation acts on.

3. Purpose
Why the operation is being performed.

4. Stage
Where the operation tends to occur in the reasoning or execution process.

5. Control Flow
Whether the phrase indicates sequencing, branching, looping, termination, continuation, or none.

## Principle

Role is primary. Object overlap alone must never create semantic equivalence.
