# Archetype Candidates

Status: Working draft

These archetypes are provisional. They should be revised as the census grows.

## Diagnostic

Primary question:
Why did this happen?

Examples:

- Five Whys
- RCA
- Fishbone
- FTA
- STAMP
- EDF

## Situational Assessment

Primary question:
What is the current state?

Examples:

- SWOT
- PESTLE
- Porter's Five Forces
- Wardley Mapping
- Business Model Canvas

## Decision / Prioritization

Primary question:
Which option should we choose or prioritize?

Examples:

- Decision Matrix
- Weighted Decision Matrix
- MoSCoW
- Eisenhower Matrix
- Kepner-Tregoe

## Planning / Alignment

Primary question:
How do we align effort toward desired outcomes?

Examples:

- OKRs
- SMART Goals
- Hoshin Kanri
- Balanced Scorecard

## Execution / Flow

Primary question:
How do we reliably perform and manage work?

Examples:

- Scrum
- Kanban
- GTD
- ITIL

## Design / Discovery

Primary question:
What should we create and why?

Examples:

- Design Thinking
- Double Diamond
- Jobs To Be Done
- Lean Startup

## Governance / Control

Primary question:
Who has authority, accountability, and control?

Examples:

- RACI
- DACI
- COSO
- COBIT

## Learning / Evaluation

Primary question:
Has learning or quality occurred?

Examples:

- Bloom's Taxonomy
- Kolb
- Kirkpatrick
- CIPP
