# What Is a Framework?

Status: Working draft

A framework is a structured knowledge artifact that guides reasoning by defining concepts, relationships, boundaries, and outputs for a class of problems.

A framework should help practitioners think more clearly.

It should not replace judgment.

## Operational Criteria

A knowledge artifact is likely a framework if it:

- defines reusable concepts
- guides reasoning or judgment
- applies across multiple cases
- produces structured outputs
- can be used without one specific software tool
- can be evaluated for coherence and usefulness

A knowledge artifact may not be a framework if it is primarily:

- a notation
- a data format
- a protocol
- a reference model
- a classification system
- a software tool
- a compliance standard
