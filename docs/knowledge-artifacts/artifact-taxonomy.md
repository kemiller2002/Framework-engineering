# Knowledge Artifact Taxonomy

Status: Working draft

## Framework

Purpose:
Guide structured reasoning.

Produces:
A reusable reasoning structure.

Common failure mode:
Confused with a methodology, checklist, or notation.

## Model

Purpose:
Represent some aspect of reality.

Produces:
An abstraction.

Failure mode:
Mistaking the model for reality.

## Methodology

Purpose:
Describe how work is performed.

Produces:
Repeatable practice.

Failure mode:
Dogmatism or ritualized use.

## Process

Purpose:
Sequence work.

Produces:
Flow.

Failure mode:
Optimizing compliance over outcomes.

## Taxonomy

Purpose:
Organize things into categories.

Produces:
Classification.

Failure mode:
Inferring relationships not actually present.

## Ontology

Purpose:
Define concepts and relationships.

Produces:
Semantic structure.

Failure mode:
Over-specification.

## Standard

Purpose:
Establish consistency or conformance.

Produces:
Rules or requirements.

Failure mode:
Stifling judgment or innovation.

## Notation

Purpose:
Represent information.

Produces:
Syntax.

Failure mode:
Confusing representation with meaning.

## Pattern

Purpose:
Capture recurring successful solutions.

Produces:
Reusable guidance.

Failure mode:
Cargo cult application.

## Reference Architecture

Purpose:
Provide reusable structural guidance.

Produces:
Architecture model.

Failure mode:
Overgeneralization.

## Tool

Purpose:
Perform or assist work.

Produces:
Execution support.

Failure mode:
Tool mistaken for the method.
